## ⚙️ 3. 핵심 방법론 (Core Methodology)

### 3.1. 데이터 정제 및 EDA (Data Cleaning & EDA)

**1단계: 데이터 신뢰도 확보 (어뷰징 탐지)**  
- 클릭 과다, 재참여 중복, CTIT 이상치 등 **10개 이상치 탐지 로직 직접 도출 및 적용**  
- 연간 **N억 원 잠재적 손실 방지 기대 (비용 절감 효과)**

**2단계: 데이터 표준화 (Feature Engineering)**  
- **11개 광고 도메인 분류**, 사용자 행동 깊이 기반 **3단계 광고 단계 분류**

**3단계: 가설 기반 인사이트 도출**  
- 성과 패턴 기반 **3가지 매체 유형 정의:**  
  `Ace`, `Multi`, `Hidden Gem` 매체 구분

---

### 3.2. 신규 광고 최적 매체 추천 예측 모델링 (Modeling)

| 항목 | 내용 |
|------|------|
| **문제 해결** | 신규 캠페인 기획 시 목표 전환수 부재 및 예산 설정 어려움 해소 |
| **추천 기준** | 전환율 기반이 아닌 **순위 지표 최적화 매체 추천** |
| **모델 구성** | `LGBRanker` (매체 추천 순위 예측) + `LightGBM` (순위 내 전환율 예측) |
| **모델 성능** | 광고별 Partition 적용 시 **전체 모델 대비 성능 개선 확인** |
